def decorate(**options):
    """Decorate the current axes.
    
    Call decorate with keyword arguments like
    decorate(title='Title',
             xlabel='x',
             ylabel='y')
             
    The keyword arguments can be any of the axis properties
    https://matplotlib.org/api/axes_api.html
    """
    ax = plt.gca()
    ax.set(**options)
    
    handles, labels = ax.get_legend_handles_labels()
    if handles:
        ax.legend(handles, labels)

    plt.tight_layout()

def underride(d, **options):
    """Add key-value pairs to d only if key is not in d.

    d: dictionary
    options: keyword args to add to d
    """
    for key, val in options.items():
        d.setdefault(key, val)

    return d

def plot_vector2d(vector2d, origin=[0, 0], **options):
    """
    
    Modified from: https://github.com/ageron/handson-ml3/blob/main/math_linear_algebra.ipynb
    """
    underride(options, head_width=0.1, head_length=0.15, length_includes_head=True)
    return plt.arrow(origin[0], origin[1], vector2d[0], vector2d[1], **options)

Three species of penguins are represented in the dataset: Adelie, Chinstrap and Gentoo, as shown in this illustration (by Allison Horst, available under the CC-BY license):

In this dataset we are told that there are three species, and we are told which species each penguin belongs to. But for purposes of clustering, we’ll pretend we don’t have this information and we’ll see whether the algorithm “discovers” the different species.

The measurements we’ll use are:

  • Body Mass in grams (g).

  • Flipper Length in millimeters (mm).

  • Culmen Length in millimeters.

  • Culmen Depth in millimeters.

If you are not familiar with the word “culmen”, it refers to the top margin of the beak, as shown in the following illustration (also by Allison Horst):
